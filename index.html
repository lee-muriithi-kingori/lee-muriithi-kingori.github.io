02 / Currently building

Open work, by priority.

  1. 01

    LestraOS

    wip

    Custom x86_64 OS · from scratch

    Bootloader, kernel, drivers, libc, userspace, package manager, desktop. Currently: VGA text, PS/2 keyboard with full scancode tables and a ring buffer, COM1 serial, PIT timer, custom libc, working shell. Next on the list: paging, scheduler, VFS, syscalls.

    C · x86_64 ASM · GNU Make · QEMU · GRUB

    ↗
  2. 02

    Rox2

    shipped

    Root-hider for Magisk / KernelSU / APatch

    Default-deny allowlist and a real Zygisk native layer using unshare(CLONE_NEWNS) and umount2 per app. Ships with a Rox-Boot companion module that bind-mounts a clean /proc/cmdline and strips ro.boot.verifiedbooterror* props. No fake attestation, no fake keybox, no fake passmark.

    C++ · Shell · CMake

    ↗
  3. 03

    Lespoof

    shipped

    LSPosed module · Samsung integrity restore

    Restores device integrity on rooted Samsungs. Hooks the system and third-party APIs so they perceive stock hardware security states. Targets Android 12+, verified against Play Integrity.

    Kotlin · Xposed

    ↗
  4. 04

    lestramk-agent-skills

    shipped

    AI agent skill library

    Production-grade collection of 22 skills for the open Agent Skills standard. Each skill ships with a SKILL.md, a setup script, and a test fixture. Covers reasoning, execution, safety, recovery, and meta-cognition.

    Markdown · Shell

    ↗
  5. 05

    RxO

    wip

    AI training orchestrator

    The Lestra AI Orchestrator. Three-repo architecture managing text-to-binary training, evaluation, and pattern recognition. Self-writes its training corpus on a single GPU.

    Python · PyTorch · CUDA

    ↗
  6. 06

    aamt-lestramk

    beta

    CLI client · talent marketplace

    Official CLI for Africa’s Vetted Talent Marketplace. Account, resumes, jobs, batch operations. Designed for power users who script their work — pipe applications, automate scans.

    TypeScript · Commander.js
